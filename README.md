# 🌤️ AI Smart Weather App

A modern weather web application that shows real-time weather, 24-hour forecast, 7-day forecast, and AI-based weather suggestions using OpenWeatherMap API.

---

## 🚀 Features

- Live weather search by city
- Current temperature, humidity, wind, pressure, visibility
- 24-hour forecast (3-hour interval data)
- 7-day forecast (approximation using API data)
- AI-powered weather suggestions (rule-based logic)
- Beautiful animated UI with glassmorphism design
- Responsive for mobile and desktop

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Glassmorphism + Animations)
- JavaScript (ES6)
- OpenWeatherMap API
